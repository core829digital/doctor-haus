## GEO/AEO Strategy

Key principles from research:
1. AI assistants decompose user queries into sub-questions and fetch each separately
2. Content must work at paragraph/block level, not just page level
3. Definition sentences in first 40-60 words of each section
4. H2/H3 as real questions users ask AI
5. Concrete data/numeric claims, not vague statements
6. Third-party corroboration (reviews, press, directory listings)
